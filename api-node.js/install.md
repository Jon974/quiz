# Installation de Node.js

*Pour pouvoir travailler sur l'API Node.js, vous devez au préalable installer Node.js sur votre machine.
  
<a href="https://openclassrooms.com/courses/des-applications-ultra-rapides-avec-node-js/installer-node-js" target="_blank">Voir un tutoriel d'installation</a>

* Notre code se trouve principalement dans le fichier serveur.js. Afin de le lancer, taper la commande suivante dans le terminal node js :

 ```
 node serveur.js
 
 ```
